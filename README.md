# PDF Merger

A Python application that allows users to merge multiple PDF files into a single PDF document.

## Features
- Merge multiple PDF files
- Easy-to-use interface
- Fast and lightweight
- Preserves original PDF quality

## Technologies Used
- Python
- PyPDF2 / pypdf
- Tkinter 

## Usage
1. Select the PDF files.
2. Arrange them in the desired order.
3. Click Merge.
4. Save the merged PDF.
